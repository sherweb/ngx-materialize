Progress

import { MzProgressModule } from 'ngx-materialize'
Determinate

Using percentage input property you can define the percentage of completion of the progress bar.

Indeterminate

When percentage input property is not provided on mz-progress tag the progress bar will be shown as indeterminate.

Colors

Optionally, colors can be customized using backgroundClass and/or progressClass input properties but are not required.

HTML Structure
<mz-progress [backgroundClass]="'blue lighten-4'" [percentage]="55" [progressClass]="'blue'"> </mz-progress>
