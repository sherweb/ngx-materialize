Textarea

import { MzTextareaModule } from 'ngx-materialize'

Ngx-Materialize leverage the use of the native HTML textarea by using the mz-textarea directive directly on the textarea element tag to ease the fact that you might use third party library that would ask you to add properties directly on that native textarea element.

Basic textarea

Label and placeholder can be combined using both label and placeholder properties.

To use the floating label directly as a placeholder provide only the label property.

You can omit the floating label and use only a placeholder by providing only the placeholder property.

Inline textarea
Textarea can be inline:

To show the textarea inline set inline property to true on mz-textarea-container component.

Disabled textarea

Disable the textarea by using the native property disabled.

Icon Prefixes

To use an icon prefix add the icon inside the mz-textarea-container component with the mz-textarea-prefix directive on the icon tag.

Character Count

You can add a character count on textarea focus using the native property length with the number of restricted characters.

HTML Structure

The textarea element must be contained inside the mz-textarea-container component otherwise an error will be shown in the console.

<mz-textarea-container> <i mz-icon-mdi mz-textarea-prefix [icon]="'message-reply-text'"> </i> <textarea mz-textarea [label]="'Comment'" id="comment-textarea" placeholder="Enter your comment" length="150"></textarea> </mz-textarea-container>
