Spinner

import { MzSpinnerModule } from 'ngx-materialize'
Sizes

Default size is medium but you can use size input property with small or big value to adjust the size accordingly.

Colors

Optionally, spinner color can be customized using color input property with red, green, blue or yellow value.

HTML Structure
<mz-spinner [color]="'blue'" [size]="'big'"> </mz-spinner>
