Chip input

Type your tag text and press enter to add tags. You can delete them by either clicking on the close icon or using your delete button.

Chip input initial value

mz-chip initial value can be set using ngModel or when bind to a formControl.

Chip input placeholders

You can specify placeholders by using placeholder input property and/or secondaryPlaceholder property for secondary placeholder when adding additional chips.

Chip input autocomplete

You can configure autocomplete with autocompleteOptions input properties.

Chip input events

mz-chip-input exposes add, delete and select output events for hooking into chips functionality.

Working with NgControl
Set chip input value

To assign a value to the mz-chip-input component, you need to provide an Array of Materialize.ChipDataObject either with NgModel directive or when setting value to a FormControl when using Reactive Forms.

IMPORTANT! As the provided value must be an array and that Angular change detection compare objects by reference, you will need to provide a new instance of the array for the mz-chip-input to be correctly updated.

example.component.html

<mz-chip-input [placeholder]="'Enter a tag'" [secondaryPlaceholder]="'+Tag'" [(ngModel)]="chips"> </mz-chip-input> <button mz-button (click)="addChip()">Add</button> <button mz-button (click)="removeChip()">Remove</button>

example.component.ts

public chips: Materialize.ChipDataObject[] = [ { tag: 'Banana' }, { tag: 'Kiwi' }, { tag: 'Mango' }, ]; addChip() { // deconstruct previous array value and add a new item to create a new // array and force change detection to update mz-chip-input component this.chips = [ ...this.chips, { tag: 'Papaya' }, ]; } removeChip() { // removes last value of array this.chips.pop(); // deconstruct previous array value to create a new array and // force change detection to update mz-chip-input component this.chips = [ ...this.chips, ]; }
Get chip input value

The returned value of the mz-chip-input component will be an array of Materialize.ChipDataObject.
