Switch

import { MzSwitchModule } from 'ngx-materialize'

Ngx-Materialize leverage the use of the native HTML input by using the mz-switch directive directly on the input element tag to ease the fact that you might use third party library that would ask you to add properties directly on that native input element.

Basic switch

Switch labels can be set using on and off properties but are not mandatory.

Disabled switch

Disable the input by using the native property disabled.

HTML Structure

The switch element must be contained inside the mz-switch-container component otherwise an error will be shown in the console.

<mz-switch-container> <input mz-switch id="switch-id" [off]="'Off'" [on]="'On'" type="checkbox"> </mz-switch-container>
