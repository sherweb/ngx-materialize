Tooltip

import { MzTooltipModule } from 'ngx-materialize'
Basic tooltip

By default tooltips will be positioned bottom when rolling over the element

Positioned tooltip

You can control the position of the tooltip by providing the position property with either bottom, top, left or right value

Delayed tooltip

The delay input property will allow you to set the delay in millisecond before the tooltips appears (1000ms in this example)

HTML enabled tooltip

You can write custom HTML inside the tooltip property by setting html input property to true

Playground

The input properties of the mz-tooltip directive makes the tooltip completely dynamic. Change the configuration below and see how the tooltip reacts.

HTML Structure

The mz-tooltip directive can be placed on any HTML element.

<button mz-button mz-tooltip [tooltip]="'I am a custom <b class=\'red-text\'>HTML</b> tooltip'" [position]="'top'" [html]="true" [delay]="1000"> Tooltip </button>
