Toast

import { MzToastModule } from 'ngx-materialize'

Materialize provides a toast implementation that will be automatically placed and sized according to the device's screen resolution. Toasts are programmatically triggered using the service MzToastService and can be customized through the parameters of the method show.

Basic toast

Basic toast using a custom message with a 4000 ms delay

Styled toast

Toasts can be customized by providing a class name to the MzToastService

Custom toast

You can pass an HTML string as the first argument as well as shown in this example where we pass in text a flat button. If you call an external function instead of in-line JavaScript, you will not need to escape quotation marks.

Callback toast

You can tell the toast to execute a callback function when it has been dismissed

HTML Structure

To trigger a toast you must use the service MzToastService and invoke show method with the desired parameters. A common example is to add the click event to a button to trigger the toast as shown below.

example.component.html

<button mz-button (click)="showToast()">Toast</button>

example.component.ts

constructor( private toastService: MzToastService, ) { } showToast() { this.toastService.show('I am a toast!', 4000, 'green', () => alert('Toast has been dismissed')); }
Service methods

Toasts can be customized using the following parameters on the method show of the service ToastService.

show(message: string, displayLength: number, className?: string, completeCallback?: Function): void
