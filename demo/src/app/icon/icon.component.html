Icon

import { MzIconModule, MzIconMdiModule } from 'ngx-materialize'

The mz-icon directive support Material Icons and the mz-icon-mdi directive Material Design Icons. Its use is not mandatory, but it provides an easy icons integration.

You may use one, both, or neither of these implementation of icons. Although we suggest you to use Material Design Icons due to the fact that not only it provides access to all the Google icons but it has a lot of community added icons that might be usefull.

Material Icons
https://material.io/icons/

icon property as cloud

size property as medium

HTML Structure
<i mz-icon [align]="'right'" [icon]="'cloud'" [size]="'medium'"> </i>
Material Design Icons
https://materialdesignicons.com/

icon property as sword

class property as red-text

size property as 36px

icon property as sword and flip to h (horizontal)

icon property as sword and rotate to 180 degree

Link

mz-icon-mdi directive added to a tag

HTML Structure
<i mz-icon-mdi [align]="'right'" [flip]="'v'" [icon]="'puzzle'" [rotate]="'45'" [size]="'36px'"> </i>
