Fixed navigation

The side navigation on the left of this demo application is an example of implementation of the mz-sidenav component with fixed property set to true. On small resolution the side navigation will be automatically hidden so you need to add a collapse button to allow the sidenav to be toggled.

Hidden navigation

The side navigation will be hidden on all screen resolution if fixed property is either omitted or false on the mz-sidenav component. To be able to toggle the side navigation visibility add a mz-button component and supply the button id trought the collapseButtonId property of the mz-sidenav component.

Collapse Button

To toggle the side navigation add a mz-button component and supply the button id to the mz-sidenav component trought the collapseButtonId property. To avoid the collapse button to be visible at the same time as a fixed side navigation on large resolution, add the class hide-on-large-only to the mz-button element.

Dropdown variation

You can add collapsible menus to your sidebar by using the mz-sidenav-collapsible component.

<mz-sidenav> <mz-sidenav-collapsible [onClose]="closeFunctionCallback" [onOpen]="openFunctionCallback"> <mz-sidenav-collapsible-header> <i mz-icon-mdi [icon]="'menu-right'" class="caret"></i>Collapsible </mz-sidenav-collapsible-header> <mz-sidenav-collapsible-content> <mz-sidenav-link> <a href="#" class="waves-effect">First Link</a> </mz-sidenav-link> <mz-sidenav-link> <a href="#" class="waves-effect">Second Link</a> </mz-sidenav-link> </mz-sidenav-collapsible-content> </mz-sidenav-collapsible> </mz-sidenav>
Open/close programmatically

Although the collapsible button way has been created to facilitate the use of the sidenav for the common "side of the page menu" use-case, you can also access the opened state of the sidenav and change its boolean value from false to true (or the other way around) to open or close it programmatically. To do so you will need to add a template reference variable on the mz-sidenav and then change the value of the opened boolean property.

sidenav.opened = {{ sidenav.opened }}
<mz-sidenav #sidenav [id]="'sidenav-demo'"> ... </mz-sidenav> <button mz-button (click)="sidenav.opened = !sidenav.opened"> Open </button>
