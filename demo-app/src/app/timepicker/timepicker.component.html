Timepicker

Ng2-Materialize leverage the use of the native HTML input by using the mz-timepicker directive directly on the input element tag to ease the fact that you might use third party library that would ask you to add properties directly on that native input element.

Basic timepicker

Label and placeholder can be combined using both label and placeholder properties.

To use the floating label directly as a placeholder provide only the label property.

You can omit the floating label and use only a placeholder by providing only the placeholder property.

Inline timepicker
Timepicker can be inline:

To show the input inline set inline property to true on mz-timepicker-container component.

Disabled timepicker

Disable the timepicker input by using the native property disabled.

Icon Prefixes

To use an icon prefix add the icon inside the mz-timepicker-container component with the mz-input-prefix directive on the icon tag.

Timepicker options

To customize the behavior of a timepicker you can use the options input property and provide a Options object with the mz-timepicker directive.

Materialize uses ClockPicker library to create the materialized time picker. You can find a complete list of available options with how to use it by following this link: https://github.com/weareoutman/clockpicker#options.
public timepickerOptions: Pickadate.TimeOptions = { default: 'now', // Set default time: 'now', '1:30AM', '16:30' fromnow: 0, // set default time to * milliseconds from now (using with default = 'now') twelvehour: false, // Use AM/PM or 24-hour format donetext: 'OK', // text for done-button cleartext: 'Clear', // text for clear-button canceltext: 'Cancel', // Text for cancel-button autoclose: true, // automatic close timepicker ampmclickable: true, // make AM PM clickable aftershow: () => alert('AfterShow has been invoked.'), // function for after opening timepicker };

HTML Structure

The input element must be contained inside the mz-timepicker-container component otherwise an error will be shown in the console.

<mz-timepicker-container class="col s12"> <i mz-icon-mdi mz-input-prefix [icon]="'clock'"> </i> <input mz-timepicker [label]="'Lunch time'" [options]="options" id="timepicker" placeholder="Select lunch time" type="text"> </mz-timepicker-container>
