Properties
mz-collapsible

Collapsibles can be customized using the following properties on the mz-collapsible element.

Property Mandatory Type Description Default value
mode string Collapsible behavior style, could be accordion or expandable accordion
onClose Function Function that will be called when a collapsible section is closed
onOpen Function Function that will be called when a collapsible section is opened
popout boolean Add popout animation when an item is clicked to show its section false
mz-collapsible-item

mz-collapsible-item will transclude any contents.

Property Mandatory Type Description Default value
active boolean Preopen the item section from the collapsible false
