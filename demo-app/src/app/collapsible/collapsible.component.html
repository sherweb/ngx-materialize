Properties
mz-collapsible

Collapsibles can be customized using the following properties on the mz-collapsible element.

mz-collapsible-item

mz-collapsible-item will transclude any contents.

Methods

The mz-collapsible component expose methods to programmatically interact with the collapsible sections. To use them you need to add a template reference variable on the mz-collapsible element.

open(index: number): void
Parameter Type Description
index number The zero base index of collapsible section to open
close(index: number): void
Parameter Type Description
index number The zero base index of collapsible section to close
