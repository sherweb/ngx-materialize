Basic pagination

By setting itemsPerPage and totalItems input property on mz-pagination component, the number of available pages will automatically be calculated.

Active page

Use currentPage input property to select a specific page when rendering.

Displayed pages

Optionally, you can change the number of displayed pages by using maxPageButton proprety. When no value is provided the number of displayed pages is defaulted to 5.

First and last page button

You can show first page and last page buttons by setting enableFirstAndLastPageButtons input property to true.

Listen to page change

You can use the ouput property changePage to call a function when page changed.

Playground

HTML Structure
