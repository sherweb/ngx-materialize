Radio Button

Ng2-Materialize leverage the use of the native HTML input by using the mz-radio-button directive directly on the input element tag to ease the fact that you might use third party library that would ask you to add properties directly on that native input element.

Basic radio button

The radio button text is set with label property. To group radio buttons add the native name property with the same corresponding value for each of the radio buttons in the group.

Gap radio button

Optionally you can use the gap styling on the radio button by setting withGap property to true.

HTML Structure

The input element must be contained inside the mz-radio-button-container component otherwise an error will be shown in the console.

<mz-radio-button-container> <input mz-radio-button [label]="'Radio text'" [withGap]="true" id="radio-button-id" name="radio-group" type="radio"> </mz-radio-button-container>
Properties

Radio buttons can be customized using the following properties on the input element when adding mz-radio-button directive.

Property Mandatory Type Description Default value
id string Id of the input
label string Radio button label
name string Radio group name
withGap boolean Show gap style radio-button false
