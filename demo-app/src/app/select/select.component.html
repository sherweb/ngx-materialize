Select

Ng2-Materialize leverage the use of the native HTML select by using the mz-select directive directly on the select element tag to ease the fact that you might use third party library that would ask you to add properties directly on the native element. Take note that the select must be contained inside a mz-select-container component otherwise an error will be shown in the console.

Basic select

Label and placeholder can be combined using both label and placeholder properties.

You can omit the floating label and use only a placeholder by providing only the placeholder property.

Multiple select

You can add the property multiple to get the multiple select and select several options

Option groups

The directive supports optgroup elements to regroup options

Circle images

You can add icons to your options in any type of select. In the option you add the image source with the data-icon attribute. You can also add the circle class to your icon.

By default the icon will be aligned on the right but you can add the left class to align your icon to the left of the select.

Disabled select

Disable the input by using the native property disabled.

HTML Structure

The select element must be contained inside the mz-select-container component otherwise an error will be shown in the console.

<mz-select-container> <select mz-select [label]="'Label'" placeholder="Placeholder"> <optgroup label="Group 1"> <option data-icon="assets/select-icon-1.jpg" class="left circle">Option 1</option> <option data-icon="assets/select-icon-2.jpg" class="left circle">Option 2</option> </optgroup> <optgroup label="Group 2"> <option data-icon="assets/select-icon-3.jpg" class="left circle">Option 3</option> </optgroup> </select> </mz-select-container>
Properties

Selects can be customized using the following properties on the select element when adding mz-select directive.

Property Mandatory Type Description Default value
label string Floating label
placeholder string Show placeholder
multiple boolean Allow to select multiple elements false
disabled boolean Disable the select false
