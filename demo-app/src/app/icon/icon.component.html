Icon

The mz-icon component support both Material Icons and Material Design Icons. Its use is not mandatory, but it provides an easy icons integration.

You may use one, both, or neither of these implementation of icons. Although we suggest you to use Material Design Icons due to the fact that not only it provides access to all the Google icons but it has a lot of community added icons that might be usefull.

Material Icons
https://material.io/icons/

icon property as cloud

iconClass property as medium

HTML Structure
<mz-icon [icon]="'cloud'" [iconClass]="'medium'"> </mz-icon>
Material Design Icons
https://materialdesignicons.com/

mdiIcon property as sword

iconClass property as red-text

mdiIconSize property as 36px

HTML Structure
<mz-icon [mdiIcon]="'puzzle'" [mdiIconSize]="'36px'"> </mz-icon>
Properties

Icons can be customized using the following properties on the mz-icon element.

Property Mandatory Type Description Default value
icon or mdiIcon string The name of the icon
iconAlign string Alignement of the icon
iconClass string Icon css class to be applied on the icon
mdiIcon or icon string The name of the material design icon (mdi)
mdiIconSize string Icon size to be applied on the icon, could be 24px, 36px, or 48px 24px
