Badge

Basic badge

Badge doesn't have any particular style by default. Only the value is displayed.

Customized badge captions

Setting the new property to true will give a background to the badge.

Badge background color can be set in the badgeClass property. All of the materialize color classes are available.

Custom caption can be used for the badge using the caption property.

Custom caption can be used in combinaison with a custom background.

HTML Structure
<mz-badge [value]="4" [caption]="'My caption'" [badgeClass]="'blue'" [new]="true"> </mz-badge>
