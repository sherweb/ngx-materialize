Properties

Dropdown can be customized using the following properties on the mz-dropdown element.

Property Mandatory Type Description Default value
align string Choose the edge the menu is aligned to could be left or right left
belowOrign boolean Show the drowdown below the button activator false
constrainWidth boolean Display the ConstrainWidth to the size of the dropdown button activator true
dropdownButtonId string Id of the dropdown button
gutter number Defines the spacing from the aligned edge 0
hover boolean Open the dropdown on hover false
id string Id of the dropdown element
inDuration number Duration of the transition enter in milliseconds 300
outDuration number Duration of the transition out in milliseconds 225
stopPropagation boolean Stops the event propagating from the dropdown origin click handler false
