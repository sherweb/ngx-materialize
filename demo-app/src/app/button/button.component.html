Properties

Buttons can be customized using the following properties on the mz-button element.

Property Type Description Default value
autofocus boolean Button will be auto focused false
buttonClass string Button css class to be applied on the button
disabled boolean Button will be disabled false
flat boolean Button will have flat style false
float boolean Button will have float style false
id string Id to be applied on the button
large boolean Button will have large style false
submit boolean Button will have a submit type false
textClass string Text class to be applied on the button text
onClick Function Function that will be called on click
