Dismissable Content

Set dismissable property to true on an mz-collection-item element to enable the collection item to be swiped away. This is only for touch enabled devices.

<mz-collection> <mz-collection-item [dismissable]="true"> Collection item 1 <a href="#!" mz-secondary-content><i mz-icon-mdi [icon]="'send'"></i></a> </mz-collection-item> <mz-collection-item [dismissable]="true"> Collection item 2 <a href="#!" mz-secondary-content><i mz-icon-mdi [icon]="'send'"></i></a> </mz-collection-item> <mz-collection-item [dismissable]="true"> Collection item 3 <a href="#!" mz-secondary-content><i mz-icon-mdi [icon]="'send'"></i></a> </mz-collection-item> <mz-collection-item [dismissable]="true"> Collection item 4 <a href="#!" mz-secondary-content><i mz-icon-mdi [icon]="'send'"></i></a> </mz-collection-item> </mz-collection>
Links collection
Collection item 1 Collection item 2 Collection item 3 Collection item 4

Alternatively, to have a list of link objects you can use an HTML anchor element and add mz-collection-link directive.

Set active property to true on mz-link-collection-item to highlight the list item.

<mz-collection> <a mz-collection-link href="#!">Collection item 1</a> <a mz-collection-link [active]="true" href="#!">Collection item 2</a> <a mz-collection-link href="#!">Collection item 3</a> <a mz-collection-link href="#!">Collection item 4</a> </mz-collection>
