Checkbox

Ng2-Materialize leverage the use of the native HTML input by using the mz-checkbox directive directly on the input element tag to ease the fact that you might use third party library that would ask you to add properties directly on that native input element.

Basic checkbox

Checkboxes use outline styling by default. Use label property to provide a clickable label to the checkbox.

Filled checkbox

To use the filled styling on the checkbox set filledIn property to true.

Disabled checkbox

Disable the checkbox by using the native property disabled.

HTML Structure

The input element must be contained inside the mz-checkbox-container component otherwise an error will be shown in the console.

<mz-checkbox-container> <input mz-checkbox [label]="'Label'" id="checkbox-id" type="checkbox"> </mz-checkbox-container>
Properties

Checkboxes can be customized using the following properties on the input element when adding mz-checkbox directive.

Property Mandatory Type Description Default value
id string Id of the checkbox
label string Label text
filledIn boolean Show filled in checkbox style false
